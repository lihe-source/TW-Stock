⚙️ GitHub Actions 設定教學
步驟一：申請 FinMind Token（選用）
FinMind Token 讓 GitHub Actions 取得完整指標（技術+基本+籌碼面）。
不設定則只計算 TWSE 免費技術指標。
前往 finmindtrade.com 免費註冊後，至個人資料複製 Token。
步驟二：GitHub Secrets 設定
1. GitHub Repo → Settings → Secrets and variables → Actions
2. New repository secret
3. Name：FINMIND_TOKEN
4. Value：貼上 FinMind Token → Save
步驟三：手動觸發第一次建置
1. GitHub Repo → Actions 分頁
2. 左側選「每日更新台股資料」
3. Run workflow → Run workflow
4. 等待約 10~30 分鐘後 data/screener.json 自動更新
